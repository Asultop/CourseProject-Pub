# 描述
给定两个整数 $a$ 和 $b$，请你依次计算并输出 $a+b$、$a−b$、$a×b$ 的结果。
注意：$$a,b$$ 不含前导零。
# 输入格式
第一行包含整数 $a$。
第二行包含整数 $b$。
# 输出格式
第一行输出 $a+b$ 的结果。
第二行输出 $a−b$ 的结果。
第三行输出 $a×b$ 的结果。

# 数据范围
|a|,|b|<$10^400$

# 输入样例：
20000000000000000
4000000000000000
# 输出样例:
24000000000000000
16000000000000000
80000000000000000000000000000000
